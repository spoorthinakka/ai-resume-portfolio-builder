The following file lists all the dependencies required to run the AI Resume & Portfolio Builder.

Installation
------------
Install all dependencies using:
    pip install -r requirements.txt

Alternatively, install them individually:
    pip install streamlit
    pip install huggingface_hub
    pip install python-docx
    pip install fpdf
    pip install python-dotenv

Purpose of Each Library
-----------------------
streamlit         : Builds the web-based user interface.
huggingface_hub   : Connects with the Hugging Face Inference API.
python-docx       : Exports resumes as editable Word (.docx) files.
fpdf              : Generates PDF resumes.
python-dotenv     : Loads environment variables such as HF_TOKEN securely.
